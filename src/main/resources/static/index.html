Spring Events & Listeners

This code example simulates part of a web blog site. Specifically, it allows posting new posts and commenting existing posts.

Do not expect to view the posts or the comments though. It is not relevant to this code example; all we want to see is how events are being published or listened to.

In addition to custom events, the example contains also listeners to existing system events provided by Spring: ContextRefreshedEvent and RequestHandledEvent (there are more).
